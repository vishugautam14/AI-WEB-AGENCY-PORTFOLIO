
# Bright Future Academy – Coaching Institute Website Demo

This project is a demo website created for a local coaching institute to help them attract student admissions online.

The goal of this website is to show how coaching institutes can improve visibility on Google and allow parents and students to easily contact them through WhatsApp and phone.

## Features

- Admissions Open homepage banner
- Courses section (IIT-JEE, NEET, Classes 9–12)
- Faculty introduction section
- Student results section
- Testimonials section
- Photo gallery
- Contact section with phone number and location
- WhatsApp contact button
- Mobile-friendly responsive design

## Purpose of the Project

This website was created as part of my freelancing portfolio to demonstrate how local coaching institutes can benefit from having a modern website for student admissions.

## Tools Used

- Wix AI Website Builder
- Basic website customization
- UI layout planning for education businesses

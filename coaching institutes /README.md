# Bright Future Coaching Institute Website

This project is a simple and responsive website created for a coaching institute.  
The website provides information about the institute, courses offered, teachers, and admission details.

## Project Overview
The purpose of this website is to help students and parents easily access information about the coaching institute, including available courses, faculty members, and contact details.

## Features
- Home page with introduction
- About the coaching institute
- Courses offered (Mathematics, Science, English, Computer)
- Teachers / Faculty section
- Student testimonials
- Admission / enrollment form
- Contact information

## Technologies Used
- HTML
- CSS
- JavaScript
- AI Website Builder

## Purpose of the Project
This website was created as a college project to demonstrate the use of AI tools for website development.

## 🎯 Project Objective
Created a high-end, aesthetic digital presence for a boutique coffee shop. The focus was on combining "Nature Distilled" 2026 design trends with a functional table reservation system to drive physical foot traffic.

## 🚀 Key Features
* **Table Reservation Portal:** Integrated booking calendar for seamless guest scheduling.
* **Aesthetic Bento-Grid Menu:** A visually organized display of seasonal brews and artisan bakes.
* **Visual Storytelling:** Cinematic imagery and minimalist typography to build brand atmosphere.
* **Local SEO & Maps:** Embedded Google Maps and "Click-to-Call" buttons for local customer convenience.
* **Mobile Optimized:** A smooth, app-like experience for booking on the go.

## 🛠️ Tools Used
* **Wix Vibe/Studio:** Core architecture and reservation backend.
* **Gemini AI:** Narrative copywriting and menu descriptions.
* **Canva:** Custom "Cloud Dancer" color palette and mood boarding.
